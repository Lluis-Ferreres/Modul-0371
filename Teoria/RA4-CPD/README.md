# Presentació de continguts del resultat d'aprenentatge 4 del mòdul 0371

### Resultats d’aprenentatge i criteris d’avaluació

4. Implanta maquinari específic de centres de processament de dades (CPD), analitzant-ne les característiques i aplicacions.

### Criteris d'avaluació
  4.1 Reconeix les diferències entre les configuracions de maquinari de tipus personal i empresarial.  
  4.2 Analitza entorns que requereixen implantar solucions de maquinari específiques.  
  4.3 Detalla components de maquinari específics per a solucions empresarials.  
  4.4 Analitza els requeriments bàsics de seguretat física, organització i condicions ambientals d’un CPD.  
  4.5 Implanta sistemes d’alimentació ininterrompuda i estabilitzadors de tensió.  
  4.6 Manipula correctament dispositius de maquinari per emmagatzematge i alimentació amb connexió en calent.  
  4.7 Documenta procediments, incidències i paràmetres utilitzats en la instal·lació i configuració de dispositius de maquinari.  
  4.8 Utilitza eines d’inventariat, registrant les característiques dels dispositius de maquinari.  
  4.9 Classifica i organitza la documentació tècnica, els controladors, les utilitats i els accessoris del maquinari.
